== Description ==
This plugin development and maintenance is temporarily freezed. No support will be provided until we decide to start the development process again.

"Users downloads are not adviced as this plugin is not a completed as we planned"

WP Ultimate Email Marketer is a ultimate email marketing plugin to create, send, schedule, track, automate and analyse your email marketing campaigns.
More than that it extends further as

1. Contact Management
2. List/Group Management
3. Subscribtion Management
4. Auto Prospects follow ups
5. Email click throughs
6. Statistics

The free version of  Ultimate Email Marketing Plugin have limitations on features and for a maximum of 500 mails/day can be sent. Upgrade to pro version to enjoy all fully functional features as described above without any limitations.
